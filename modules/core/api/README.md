# API folder

This folder contains the API codebase for the project. It includes the TRPC setup for client and server side, and the API routes for the project.

## init.ts

This file contains the TRPC setup for client and server side.

## router.ts

This file contains the API routes for the project.

## server.tsx

This file contains the server-side logic for the API. This will be used in React Server Components to make requests to the API.

## client.tsx

This file contains the client-side logic for the API. This will be used with Tanstack query to make requests to the API.
