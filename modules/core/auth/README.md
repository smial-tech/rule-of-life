# Auth folder

This folder contains the authentication module for the project. It includes the main logic for the authentication module, including user authentication, session management, and authorization checks. The auth is based on the `better-auth` package. Check better-auth [documentation](https://better-auth.com/docs/introduction) for more information.


### Google Auth Provider

The Google auth provider is used to authenticate users using their Google account. It is implemented using the `google` provider in the `better-auth` package. Guide to setup google auth provider [here](https://better-auth.com/docs/authentication/google)
